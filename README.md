# 🎮 UE5 Creator Studio

Unreal Engine 5.7.4 Companion App — PC & Mobiel — met Groq AI Assistent.

## 🚀 Installeren & Starten

```bash
npm install
npm run dev
```

Ga naar `http://localhost:5173`

## 🌐 Deployen op Vercel (gratis)

1. Maak een repo aan op GitHub
2. Push alle bestanden
3. Ga naar [vercel.com](https://vercel.com) → Import repo → Deploy

## 🤖 Groq AI instellen (GRATIS)

1. Ga naar [console.groq.com](https://console.groq.com)
2. Maak gratis een account aan
3. Klik op **"API Keys"** → **"Create API Key"**
4. Kopieer de key (begint met `gsk_`)
5. Klik in de app op de 🤖 knop rechtsonder
6. Plak je key en klik Opslaan

## 📦 Bestanden

```
src/
  App.jsx          ← Hele app + alle pagina's
  index.css        ← Styling
  main.jsx         ← Entry point
  components/
    AIChat.jsx     ← Groq AI chat
```
