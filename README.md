# 🎮 UE5 Creator Studio

Een volledige **Unreal Engine 5.7.4** companion app — werkt op PC én mobiel.

## ✨ Features
- **Level Editor** — Actor plaatsing, viewport, transform controls
- **Blueprint Editor** — Visueel scripting, nodes, variables
- **Material Editor** — PBR materials, node library, live preview
- **Sequencer** — Cinematic timeline, keyframes, camera tracks
- **AI & Gameplay** — Behavior Trees, Blackboard, Input bindings
- **Niagara VFX** — Particle emitters, presets, live preview
- **Audio Systeem** — Sound library, Sound Cue editor, 3D audio
- **World Building** — Landscape, PCG, sky atmosfeer, water
- **Asset Manager** — Filter, zoeken, favorieten
- **UI Builder** — UMG widget designer
- **Project Settings** — Lumen, Nanite, platforms, build config
- **🤖 AI Assistent** — Powered by Claude (Anthropic API)

## 🚀 Installeren

```bash
npm install
npm run dev
```

Open dan `http://localhost:5173` in je browser.

## 🌐 Deployen op GitHub Pages

1. Bouw de app:
```bash
npm run build
```

2. Push naar GitHub en activeer GitHub Pages vanuit de `dist/` map.

Of gebruik **Vercel** — connect je GitHub repo en deploy in één klik.

## 🤖 AI Assistent instellen

1. Ga naar [console.anthropic.com](https://console.anthropic.com)
2. Maak een API key aan
3. Klik op de 🤖 knop rechtsonder in de app
4. Voer je API key in

## 🛠 Tech Stack
- React 18
- Vite 5
- CSS Custom Properties
- Anthropic Claude API
- Google Fonts (Orbitron + Barlow)
